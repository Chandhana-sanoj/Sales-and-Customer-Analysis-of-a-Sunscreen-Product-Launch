# Power BI Dashboard

This folder contains the Power BI dashboard and visualization assets.

## Files

- Sunscreen_Product_Launch_Analysis.pbix  
  Interactive dashboard built using Power BI.

- dashboard_screenshot.png  
  Screenshot of the final dashboard used in the main README.

## Dashboard Features
- KPI cards (Revenue, AOV, Growth, Retention, Orders)
- Product performance charts
- Customer behavior insights
- Channel and regional analysis
- Interactive slicers for filtering

## Tools Used
Microsoft Power BI Desktop
